[how to build]
* pre-installation: CMake Tools by Microsoft

1. If you can't see "CMake" tools in the left panel of vscode, Press Ctrl + Shift + P.
2. And then select "CMake: Build" and you can see "CMake" tools in the left panel.
3. Pressing "CMake" tools in the left panel, you can build cmake project  
4. After running the cmake as above, You can debug it pressing F5.
